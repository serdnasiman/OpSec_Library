## Set your DNS through Cloudflare

To prevent your ISP to track your DNS queries.

### Firefox 

- 1.- Go to settings.
- 2.- General Options -> Network Settings.
- 3.- Activate the Safe DNS Option (Enable Secure DNS)
- 4.- Select Max Protection and the provider (Cloudflare, 1.1.1.1)
